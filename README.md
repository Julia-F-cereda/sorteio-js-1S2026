<h3>DESCRIÇÃO DO PROJETO</h3>
Este projeto foi desenvolvido com o objetivo de praticar a criação de elementos, objetos e metodos no utilizando javaScript, visando construir mais conhecimento e lógica.


<h3>COMO EXECUTAR</h3>
<P>1. Baixe os arquivos do projeto.</P>
<P>2. Abra o arquivo, va até a pagina index, e abra em live server.</P>
<P>3. Para abrir em liveServer basta clicar baixar o liveserver em seu visualStudio, clicar com o botao direito e escolher a opção "Open with live server".</P>

<h3>FUNCIONALIDADES DO PROJETO</h3>
<p>Escolha dos numeros utilizando uma barra</p>
<p>Sortear quantos numeros quiser, salvando os 5 ultimos em um histórico</p>
